Tools is reserved for calculators, checklists, templates, and small decision aids that support the article library.

Tools should be simple enough to audit. If a tool makes a recommendation, it should show the inputs that shaped the output.

## Start Here

- Use a checklist when a calculator would imply fake precision.
- Show assumptions beside the result.
- Link each tool back to a guide that explains the idea.
