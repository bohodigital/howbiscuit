# Link previews

Link previews are optional governed snapshots under `content/link-previews/`. A record includes the destination URL, title, description, site label, capture date, source ID, optional media ID, rights state, and status.

The source must be declared by the article. Preview media must also be declared and have registered rights. Media-free previews use `not-required`; previews with media use `registered`.

Snapshots do not imply the destination is currently available, accurate, endorsed, or commercial. Merchant destinations use the separate unpaid shopping record system.
