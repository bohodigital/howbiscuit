# LaTeX adapter

The trusted LaTeX lane is a narrow adapter for the existing salt-and-ice article. It accepts the governed subset documented in `docs/latex-article-pipeline.md`, compiles to the same normalized article model, and produces KaTeX plus MathML without arbitrary TeX execution.

LaTeX governance records bind idea, brief, approval, sources, testing, media, commerce IDs, and workflow. It shares duplicate route checks, publication surfaces, analytics rules, related articles, RSS, sitemap, `llms.txt`, and Pagefind eligibility with standard packages.

New general-purpose TeX execution is outside Handoff 2.
